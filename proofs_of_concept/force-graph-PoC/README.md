# Force-Graph Proof of Concept

Interactive phylogenetic tree visualization using react-force-graph with both 2D and 3D views.

## Features

- **2D & 3D Views**: Toggle between canvas-based 2D and WebGL 3D visualizations
- **Performance Metrics**: Load time, node count, and edge count displayed
- **Phylogenetic Trees**: Pre-loaded datasets from converted Newick edgelists
- **Interactive Navigation**: Pan, zoom, and rotate (3D mode)

## Available Datasets

- **Simulated 1K**: ~1,000 nodes (ete3 generated)
- **Simulated 3K**: ~3,000 nodes (ete3 generated)
- **Simulated 10K**: ~10,000 nodes (ete3 generated)
- **H. influenzae MLST**: ~3K nodes (PubMLST)
- **C. coli MLST**: ~3K nodes (PubMLST)

## Running the Demo

```bash
npm install
npm run dev
```

Open http://localhost:5173 and select datasets from the dropdown.

## Library Information

- **react-force-graph-2d**: Canvas-based 2D force-directed graph
- **react-force-graph-3d**: Three.js-based 3D force-directed graph
- Both use d3-force for physics simulation under the hood

## Performance Notes

- 2D mode is generally faster for rendering
- 3D mode provides better spatial separation for dense trees
- Force simulation continues until nodes stabilize (watch FPS)

## Data Format

Phylogenetic trees are loaded from CSV edgelists:

```csv
Source,Target
Node1,Node2
Node2,Node3
```

Generated by `analysis/newick_to_edgelist.py` from Newick tree files.
